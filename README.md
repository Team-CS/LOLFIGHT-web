# LOLFIGHT

> 리그 오브 레전드 길드 스크림 커뮤니티 플랫폼

## 소개

LOLFIGHT는 리그 오브 레전드 유저들을 위한 길드 기반 커뮤니티 플랫폼입니다. 길드를 생성하고 다른 길드와 스크림(연습 경기)을 매칭하며, 커뮤니티 활동을 통해 함께 성장할 수 있는 서비스를 제공합니다.

**서비스 URL:** https://lolfight.kr

## 주요 기능

### 길드 시스템
- 길드 생성 및 관리
- 길드원 모집 및 가입
- 길드 랭킹 (래더 점수 기반)

### 스크림 매칭
- 5v5 팀 구성 및 매칭
- 스크림 신청/수락/거절
- 매치 결과 기록

### LCK 프로리그
- LCK 팀 순위 조회
- 경기 일정 확인
- 승부 예측 이벤트

### 롤로세움
- 플레이 영상 업로드
- 커뮤니티 투표 판정

### 커뮤니티
- 게시판 (자유, 공지, 이벤트, 길드원 모집)
- 댓글 및 추천

### 프로필
- Riot 계정 연동
- 출석 체크 및 포인트 적립
- 상점 아이템 구매
